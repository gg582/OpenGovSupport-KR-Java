뜌땨 생활행정 (Tyutya Desktop)

본 소프트웨어는 「소득세법」, 「국민기초생활 보장법」 등 공개 법령의 산식을
결정적(deterministic)으로 평가하는 참고 자료를 제공합니다. 산출 결과는
신고·납부·수급의 효력을 갖지 않으며, 반드시 홈택스 / 복지로 / 정부24 또는
세무전문가를 통해 확정해야 합니다.

본 소프트웨어는 외부 API 를 호출하지 않으며, 모든 계산은 사용자의 컴퓨터
내부에서만 수행됩니다.

— 면책 —
산출 결과의 정확성·완전성·최신성을 보증하지 않습니다.
사례별 적용 여부와 예외 처리는 사용자가 직접 확인해야 합니다.
