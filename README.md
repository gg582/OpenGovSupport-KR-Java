# 세무·복지 법령 계산 엔진

「국민기초생활보장법」·「소득세법」·「법인세법」·「상속세 및 증여세법」·「부가가치세법」·「조세특례제한법」 등 공개 법령의 산식을 코드로 평가하는 오픈소스 엔진.

## 기능 개요

- **개인세무**: 종합소득세·근로소득공제·연말정산·의료비·교육비·월세·연금·기부금·자녀공제·단순경비율·근로장려금·법인세·상속세·증여세·부가가치세·결혼공제·체육시설공제
- **사회복지**: 사적이전소득·이자소득·재산상담·상속분·긴급공제·해외체류·공용 유틸
- **연도별 시점비교**: 2023·2024·2025·2026년 법령 데이터 동시 비교
- **계산흐름도**: 종합안내판에서 서식을 불러오기·저장·연결하여 사용자 정의 흐름 설계

## 기술 사양

- Java ≥ 21, Spring Boot, Maven
- Next.js 14, React, TypeScript
- BigDecimal 임의정밀 산술 — 부동소수점 오차 없음
- DB 없이 JSON·Java 상수로 기준값 관리 — 오프라인 실행 가능

## 실행 방법

```bash
make run            # backend :8080 + frontend :3000
make run-backend    # backend only
make run-frontend   # frontend only
make test           # 테스트
```

Docker:

```bash
cp .env.example .env
make compose-up
```

## 데스크탑 응용프로그램

Windows·macOS·Linux·FreeBSD용 데스크탑 응용프로그램을 함께 제공.
낮춤 Java + Next.js를 Electron 셸로 묶어 더블클릭 설치 가능.

## 면책

본 엔진은 세무사·세무대리 행위를 수행하지 않으며, 산출 결과의 정확성·완전성·최신성을 보증하지 않는다.
실제 신고·납부·수급은 반드시 홈택스(국세청)·복지로(보건복지부)·정부24 또는 세무전문가를 통해 확정해야 한다.
